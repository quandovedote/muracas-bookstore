# Muraca's Bookstore

A simple static webpage for a fictional bookstore, built using HTML5.  
The page features different book categories and allows users to navigate easily through sections like Action, Drama, and Contact.

## Features

- Semantic HTML5 structure
- Clean and accessible navigation
- Organized sections for each book category
- Easy-to-read layout with informative comments in the code

## Technologies

- HTML5

## Live Preview

You can view the page by opening `index.html` in your browser.
